# LiteLoaderQQNT-Plugin Masuda

仅供交流学习使用！ 通过webSocket开放QQNT接口

可以配合[Masuda.QQNT(C#)](https://github.com/ssccinng/Masuda.QQNT)二次开发

依赖于LLAPI插件

可以在config.json中修改端口号 ip等
